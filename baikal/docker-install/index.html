Installation using Docker

Although we do not provide an official Baikal Docker image there is a community maintained image ckulka/baikal available on DockerHub.

It works well with Let's Encrypt and Traefik to provide secure installations.

For details and installation instructions see ckulka/baikal-docker on GitHub.
