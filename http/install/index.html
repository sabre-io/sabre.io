Installation

sabre/http must be installed using composer.

Requirements

You will need a recent version of PHP. sabre/http currently requires PHP 5.4.

Installing with composer

If composer is not yet on your system, follow the instructions on Composer's website to install it.

From your project directory, the easiest way is then to simply call:

composer require sabre/http ~7

This rule ensures that you install the latest HTTP package in the 2.0 range of packages, but it does not install 2.1 or higher, which could result in a backwards compatibility break.

After running this, sabre/http should be installed, and you can load it in by including the autoloader:

include 'vendor/autoload.php';
