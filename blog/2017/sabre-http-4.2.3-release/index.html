sabre/http 4.2.3 released

We just released sabre/http 4.2.3. This release adds a workaround for the 4 GB filesize limitation on 32-bit systems.

To upgrade sabre/http, make sure your composer.json requires "^4.2" for sabre/http and run composer update.

Full changelog can be found on Github
